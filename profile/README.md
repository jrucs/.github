# JRUCS Reconfigurable Universal Card System (JRUCS)

You can read about generic games in the [JRUCS Rules](https://github.com/jrucs/jrucs-rules).

## Alchemy

The year is 2104. Magic is seeping into our universe. Alchemists roam the Earth battling one another.

* [Alchemy](https://github.com/jrucs/alchemy)
* [Alchemry Contrib](https://github.com/jrucs/alchemy-contrib)

## Gods

The more disciples a god has, the more powerful they are. As a minor deity, your goal is to create a religion that subsumes all others, either by killing the nonbelievers or converting them to yours, the true religion.
